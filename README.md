# AI-Based Study Planner

## Overview
AI-Based Study Planner is a smart web application that helps students create a personalized study plan using Artificial Intelligence logic. It focuses on improving productivity by prioritizing weak subjects and managing time effectively.

This project is developed for hackathons and academic purposes to solve real-world student problems.

---

## Problem Statement
Students often face:
- Poor time management
- No personalized study plan
- Difficulty focusing on weak subjects
- Stress before exams

Traditional study schedules are fixed and not adaptive.

---

## Solution
The AI-Based Study Planner generates a customized study timetable based on:
- Subjects
- Weak areas
- Available daily study time
- Exam deadlines

The planner adapts according to student needs and improves learning efficiency.

---

## Use of AI
The system uses AI-based logic to:
- Analyze student input
- Allocate more time to weak subjects
- Optimize daily study hours
- Provide smart study recommendations

This project is AI-ready and can be extended using Gemini or OpenAI APIs.
